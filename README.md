# 💻 Tareas de Programación Orientada a Objetos (POO)

Este repositorio contiene diferentes ejercicios y proyectos desarrollados como parte de mis prácticas de **Programación Orientada a Objetos (POO)**.
Los programas están implementados en **C++** y tienen como objetivo aplicar conceptos fundamentales como:

* Clases
* Objetos
* Encapsulamiento
* Métodos
* Interacción entre objetos

# 📂 Proyectos del repositorio

## 🏦 1. Gestión de Cuenta Bancaria (POO)

Programa que simula el funcionamiento básico de una **cuenta bancaria** utilizando clases.

### Funcionalidades

* Crear una cuenta
* Depositar dinero
* Retirar dinero
* Consultar saldo

### Conceptos aplicados

* Clases y objetos
* Métodos públicos y privados
* Encapsulamiento

---

## ✊📄✂️ 2. Piedra, Papel o Tijera (POO)

Implementación del juego clásico **Piedra, Papel o Tijera** utilizando programación orientada a objetos.

### Características

* Dos jugadores (usuario o computadora)
* Evaluación automática de la jugada
* Determinación del ganador

### Conceptos aplicados

* Interacción entre objetos
* Métodos de evaluación
* Lógica del juego

---

## ♜ 3. Torre de Ajedrez (POO)

Programa que simula el **movimiento de una torre en el tablero de ajedrez**.

### Funcionalidades

* Representación del tablero
* Movimiento de la torre
* Validación de movimientos válidos

### Conceptos aplicados

* Modelado de objetos
* Validación de reglas
* Uso de métodos para controlar movimientos

# ▶️ Cómo ejecutar los programas

1. Clonar el repositorio

```
git clone https://github.com/dafnesaraviapedrazas/tareas_github.git
```

2. Entrar al repositorio

```
cd tareas_github
```

3. Compilar un archivo

```
g++ archivo.cpp -o programa
```

4. Ejecutar el programa

```
./programa
```
